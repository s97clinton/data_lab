# Steven Clinton's data_lab
Steven Clinton's Data Lab Repo; Clinton is a Data Scientist who works with Python (primary) and R (secondary), this repo (in development) has end-to-end tools for everything from data engineering to predictive modeling to data visualizations.

## Current Project Directories
### NFL/nfl_model_v4
- Contains code to work with data imported from the nfl_data_py package
- Key file is 'main.py', which contains the run file for the full modeling process (in development); the workspace will be actively restructured as the project takes its full form.
- This file also contains a variety of EDA notebooks used to examine the available data.
- It also contains 'run_update_nfl_data_py.py', which calls functions to pull and store the data from nfl_data_py in a local directory.
### NFL/nfl_model_v3_R
- Contains code to run game and player projections with my "v3" NFL model built in R.
- Dependent on .csv files that are not stored in the repository.
### NFL/nfl_data_bowl
- Contains preliminary work from the 2025 NFL Data Bowl.
